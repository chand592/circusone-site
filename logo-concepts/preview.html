Three modernized marks for Circus One.

Each preserves the original's signature — an orbit around a glossy sphere, the lowercase circusone wordmark, and the split-tone text treatment — with a more detailed, faithful icon and colors closer to the original: darker metallic greys for the ring and a bolder red for the sphere.

04 Original Refreshed

The original logo, unchanged in structure — same orbit ring around a glossy sphere, same lowercase circusone wordmark, same split-tone text. Only the colors are refreshed: a bolder red for the sphere and darker metallic greys for the ring. Nothing else moved. This is the conservative option — the brand mark people already recognise, just sharpened.

01 Orbit Sphere

Most faithful to the original. A glossy bolder-red sphere with radial gradient and specular highlight, wrapped by a darker metallic-grey orbit ring that passes behind (top arc) and in front of (bottom arc) the sphere. The ring breaks at upper-left to suggest motion, as in the original mark. Wordmark keeps the original split-tone: "circus" near-black, "one" medium grey.

How to review: open this file in a browser, or serve the folder with npm run dev and visit the concepts path. Each concept has a cream version (for the site header) and a reversed cream-on-dark version (for the deep-red footer).

Note on fonts: the wordmark uses a geometric sans fallback stack (Trebuchet MS / Century Gothic / Futura). The chosen concept should be finalized with a licensed geometric sans (e.g. Montserrat or Poppins) and the text converted to outlined paths for a production-ready, portable logo file.
